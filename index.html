AI Automation n8n + OneDrive + AI + Excel

Consolidate Raw Collection Data

Built an n8n workflow that detects new files in a OneDrive folder, extracts raw collection data, compares the incoming columns against the target Excel structure, and uses AI to identify renamed, removed, or incorrect column names before appending clean records.

  • OneDrive trigger for newly uploaded collection files
  • AI-assisted column matching and correction
  • Cleaned records appended into an Excel table
PBI Automation n8n + Power BI + Microsoft Teams

Power BI Dataset Refresh Monitor

Built an n8n automation that refreshes Power BI datasets on a scheduled cadence up to 8 times per day, checks the refresh result, and sends Microsoft Teams notifications for successful and failed runs.

  • Scheduled Power BI dataset refresh workflow
  • Refresh history and status validation
  • Success and failure alerts sent to Microsoft Teams
Waybill Automation n8n + OneDrive + PDF Extraction + Excel

Waybill PDF Data Extraction

Built an n8n workflow that watches a folder for uploaded waybill PDF files, downloads the file, extracts the required waybill data, formats it with JavaScript, and appends the structured records into an Excel table.

  • Folder trigger for new waybill PDF uploads
  • Automated PDF data extraction and formatting
  • Excel table append for organized tracking
AI Forecasting n8n + AI + LightGBM + Sales Data

AI Sales Forecast

Built an n8n forecasting workflow that predicts sales using a hybrid AI and LightGBM approach. The process prepares sales history, enriches the forecast with external signals, and generates prediction results for business planning.

  • Sales history extraction and cleaning
  • Forecast inputs from weather, holidays, crisis events, and location signals
  • Hybrid AI and LightGBM prediction workflow for more accurate forecasts
Data Engineering n8n + Power BI + SQL Server + ETL

ETL from Power BI to SQL Server

Built an ETL workflow that pulls data from a Power BI dataset, prepares records for SQL Server, and performs real-time insert and update operations to keep the database synchronized.

  • Power BI dataset extraction and refresh-status handling
  • SQL Server insert and update logic for synced records
  • Operational notifications for process success or failure
